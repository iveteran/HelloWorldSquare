Mac OS常用快捷键

Command (或 Cmd) ⌘：最核心的键（类似于 Windows 的 Ctrl）。
Shift ⇧：通常用于辅助功能或反向操作。
Option (或 Alt) ⌥：用于次要功能或特殊字符。
Control (或 Ctrl) ⌃：在 Mac 上用得比 Windows 少，主要用于系统级操作。

聚焦搜索 (Spotlight)：
⌘ + Space (空格)
F4

显示桌面：
Command + F3

平铺Apps
F3

F9: 打开多媒体
F10: 静音
F11: 减小音量
F12: 增大音量

切换 App:
⌘ + Tab
按住 ⌘ 不放，点按 Tab 切换；按 Q 可直接关闭选中的 App。

强制退出:
⌥ + ⌘ + Esc
类似于 Windows 的 Ctrl+Alt+Del，当程序卡死时使用。

锁屏：
Control + Command + Q

Shift + Command + 3:
截取整个屏幕，并自动保存为文件到桌面。 

Shift + Command + 4:
截取屏幕的一部分。 按下后出现十字线，拖动选择区域后松开即可。 

Shift + Command + 4 + 空格键:
截取单个窗口或菜单栏。 将光标移到目标上高亮后点击即可。 

Shift + Command + 5:
呼出截图工具栏，提供全屏、窗口、选区截图及屏幕录制选项。 

保存到剪贴板:
在执行任何截图快捷键时，同时按下 Control 键（如 Control + Shift + Command + 4），截图会复制到剪贴板。 

取消截图:
按下 Esc 键可以取消正在进行的截图操作。 

移动截图框:
在使用 Shift + Command + 4 选择区域时，按住空格键可以移动整个选框。 

关闭当前窗口:
⌘ + W
只是关掉当前页面（如浏览器标签页），程序还在后台运行。

退出当前程序:
⌘ + Q
彻底杀掉程序进程 (Quit)。

最小化窗口:
⌘ + M
收到程序坞 (Minimize)。

隐藏当前程序:
⌘ + H
推荐！ 瞬间隐藏当前 App 所有窗口 (Hide)，桌面瞬间清爽。

全屏模式:
Ctrl + ⌘ + F
进出全屏模式。

快速预览：
选中文件后按 Space (空格)
这是 Mac 最强大的功能之一，无需打开软件即可预览图片、文档、视频。

删除文件：
⌘ + Delete

清空废纸篓：
⌘ + ⇧ + Delete

新建文件夹：
⌘ + ⇧ + N

查看属性：
⌘ + I
Get Info，查看文件大小、路径等

跳至行首/行尾
⌘ + ← / →
相当于 Win 的 Home / End。

跳至文首/文末
⌘ + ↑ / ↓
快速回到顶部或底部。

按单词移动
⌥ + ← / →
光标一次跳过一个英文单词或中文词组。

删除整行/单词
⌘ + Delete
删除光标左侧整行内容。

调出表情包
Ctrl + ⌘ + Space
快速输入 Emoji 😄。
