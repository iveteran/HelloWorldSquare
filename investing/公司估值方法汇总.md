# 股票估值实战指南：方法、行业与阶段全解析

### 一、 核心估值工具箱 (Valuation Toolkit)

对标的进行估值通常采用“相对估值”与“绝对估值”相结合的方式。

#### 1. 相对估值法 (市场比较)
通过参考市场同类资产的定价来确定目标公司的价值。

* **PE (市盈率)**：$股价 / 每股收益$。最常用，适合盈利稳定的公司。
* **PB (市净率)**：$股价 / 每股净资产$。适合重资产或周期性行业。
* **PS (市销率)**：$股价 / 每股销售额$。适合高增长但尚未盈利的公司。
* **PEG (成长调节市盈率)**：$PE / 净利润增长率$。弥补 PE 缺陷，通常 $PEG < 1$ 视作低估。
* **EV/EBITDA (企业价值倍数)**：剔除资本结构和折旧影响，适合重资产制造业。

#### 2. 绝对估值法 (内在价值)
通过预测未来现金流并折现回当前价值。

* **DCF (现金流折现)**：估值的核心模型，预测自由现金流。
* **DDM (股利贴现)**：适合高分红、低增长的公用事业。
* **NAV (净资产价值)**：计算变现价值，适合地产、资源股。
* **rNPV (风险调整净现值)**：生物医药专用，根据临床成功率调整。

#### 3. 分部估值法 (SOTP)
* **适用对象**：业务复杂的综合性集团（如腾讯、伯克希尔）。
* **方法**：将各项业务（如游戏、云服务、投资部）单独估值后加总。

### 二、 行业估值矩阵 (Industry Valuation Matrix)

不同行业的商业逻辑决定了其适用的核心估值指标。

| 行业分类 | 核心指标 | 辅助指标 | 估值逻辑与关注点 |
| :--- | :--- | :--- | :--- |
| **消费/白马股**<br>(食品饮料、家电) | **PE (市盈率)** | PEG, DCF | 利润是核心驱动力，关注净利率与品牌护城河。 |
| **科技/互联网**<br>(SaaS、电商) | **PS, PEG** | EV/EBITDA | 高成长是关键。早期看营收规模(PS)，中期看增速匹配(PEG)。 |
| **金融/银行**<br>(银行、保险) | **PB (市净率)** | EV (内含价值) | 利润调节空间大，净资产更真实。保险主要看内含价值。 |
| **周期性行业**<br>(能源、航运、有色) | **PB (市净率)** | EV/EBITDA | 盈利随宏观经济剧烈波动，PE 在高位时往往是陷阱，资产价值是底线。 |
| **生物医药**<br>(创新药) | **rNPV** | PS (市销率) | 大多亏损，主要评估研发管线 (Pipeline) 的未来潜在现金流。 |
| **房地产/REITs** | **NAV** | FFO (营运现金流) | 核心看土地储备变现价值 (NAV) 及租金现金流 (FFO)。 |

### 三、 企业生命周期估值法 (Life Cycle Valuation)

| 阶段 | 特征描述 | 推荐估值方法 | 投资逻辑 |
| :--- | :--- | :--- | :--- |
| **初创期**<br>(Start-up) | 有产品雏形，无利润，甚至无营收。 | **一级市场对标、P/MAU** | 看赛道规模 (TAM) 和想象空间。 |
| **成长期**<br>(Growth) | 营收爆发增长，可能微利或亏损。 | **PEG, PS** | 看营收增速，忽略短期利润，关注市场占有率。 |
| **成熟期**<br>(Maturity) | 增长放缓，护城河深，现金流充沛。 | **PE, DCF, 股息率** | 看盈利稳定性与分红回报，类似债券属性。 |
| **衰退期**<br>(Decline) | 行业萎缩，利润下滑。 | **PB, 清算价值** | 防止 PE 陷阱，只看资产变现价值或并购价值。 |

### 四、 估值实操三步走 (Action Plan)

1.  **定性 (Qualitative)**：
    * 确认行业属性（是看资产的银行，还是看增长的科技？）。
    * 确认生命周期（是看未来的初创，还是看现在的蓝筹？）。
2.  **定量 (Quantitative)**：
    * **纵向对比**：当前估值处于该股票历史 10 年的什么分位？（< 20% 分位通常较安全）。
    * **横向对比**：相比同行业竞争对手，是贵了还是便宜了？
3.  **安全边际 (Margin of Safety)**：
    * 计算出合理估值后，预留 **20%-30%** 的折扣作为安全垫再买入。
