# MACD指数平滑异同移动平均线

## 简介

指数平滑异同移动平均线，它是以快速及慢速两条指数移动平均线，计算两者之间的差离值(DIF)，再利用差离值与差离值平均值(DEM)的收敛(合并)与发散(分离)的征兆，从而决定股票买卖的时机。
 
## 计算公式

1. 平均指数(DI) =(最高价 + 最低价 + 2 * 收盘价)/4
2. 指数平滑移动平均数(EMA) =前日EMA + 2/(1 + 移动平均天数) * (当天平均指数 - 前日EMA)
3. 差离值(DIF) = 12日EMA - 26日EMA
4. MACD = 前日MACD + 2/(1+移动平均天数) * (DIF - 前日MACD)
 
## 应用

1. MACD及DIF均为正值，可视为多头市场。
2. MACD及DIF均为负值，可视为空头市场。
3. DIF向上突破MACD，买进讯号。
4. DIF向下跌破MACD，卖出讯号。
5. DIF值由负转正，且穿越MACD，买进讯号。
6. DIF值由正转负，且突破MACD，卖出讯号。
7. 如果MACD及DIF皆为正值，且DIF向上突破MACD，此为买方市场，做多较有利。
8. 如果MACD及DIF皆为负值，且DIF向下跌破MACD，此为卖方市场，作空较为有利。
9. DIF与大盘指数呈背离走势时，若股价连续创新低点，而DIF值并未创新低点，此为『正背离』走势，为买进时机；
   反之，若股价连续创新高点，而DIF值并未创新高点时，此为『负背离』走势，为卖出时机。
 
## 特色

1. MACD比较能掌握明显的涨升波段，大跌后找买点应该优先考虑使用MACD。MACD在测试股市主要趋势走向是一个相当良好的分析工具。
2. MACD对中长期投资买卖有独到的分析能力，对于大波段涨跌幅确认颇为准确。由MACD波动的周期性，可帮助波浪理论研究者，对波段的涨跌幅加以确认。
3. MACD可以减少52％移动平均线频频出现假突破的买卖点，减少无效的交易次数而提高获利能力。
4. 涨升一大段后，MACD与DIF经常『纠缠不清』，买进讯号变差，此时不宜使用。
